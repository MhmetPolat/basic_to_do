# Task Management Program
This is a simple task management program that allows users to add, list, and delete tasks.

## Features
- Add tasks to the task list.
- List all current tasks.
- Delete tasks from the task list.
- Exit the program.

## Usage
1. Run the script.
2. Choose an option from the menu:
   - **1** to add a task.
   - **2** to list tasks.
   - **3** to delete a task.
   - **4** to exit the program.
